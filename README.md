# Brik Launch — Venues Research Map

מפת מחקר אינטראקטיבית לקראת ההשקה של Brik: venues, שת"פים מקצועיים ומוזיאונים בסקייל.

**צפייה:** פתחו את [index.html](index.html) בדפדפן, או דרך GitHub Pages:
`https://elirazdekalowix.github.io/brik-launch-venues/`

## מה יש בפנים

- **Venues (293)** — מקומות להשקה, יחץ, תערוכות ואירועים — עם פילטרים לפי סוג, מדינה, טאגים (יוקרה, מוזרות, אימרסיבי, אינסטליישן, אירוע השקה ועוד) והתאמה לשת"פ
- **שת"פים מקצועיים (40)** — מגה-אירועים, מפיקות טקסים, אולפני חוויות, במות וטורים
- **מוזיאונים בסקייל (24)** — פוטנציאל תערוכות: Brik כאמן מציג / כנותן שירות לאמנים

## פיצ'רים

- שלוש תצוגות: כרטיסים / שורות / מפת עולם 🌍
- מועדפים ★ והערות אישיות 💬 (נשמרים בדפדפן, נכללים בייצוא)
- ייצוא CSV של כל הדאטה
- תאריכי פסטיבלים לשנה הקרובה
- "להכיר לעומק" — הרחבה על כל מקום

## להציע מקום חדש

פתחו [Issue](../../issues/new) עם שם המקום, עיר, תיאור קצר ולמה הוא מתאים ל-Brik — או לחצו על כפתור "+ הצע מקום" בתוך המפה.
